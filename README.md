# 🍽 Royal Spice Restaurant

A modern and responsive restaurant website built with **React JS** and **Vite**. The application provides a premium restaurant experience with an elegant user interface and essential restaurant features.

---

## 🚀 Features

* 🏠 Responsive Home Page
* 🍽 Menu with Recipe Details
* 🛒 Shopping Cart using React Context API
* 📅 Table Reservation System
* 👨‍🍳 Services Page
* ℹ️ About Us Page
* 📞 Contact Page
* 🔐 Login Page
* 📱 Mobile-Friendly Design

---

## 🛠 Technologies Used

* React JS
* Vite
* React Router DOM
* React Context API
* CSS3
* JavaScript (ES6)

## ▶️ Installation

```bash
git clone <repository-url>

npm install

npm run dev
```
